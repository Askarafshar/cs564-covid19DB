Following steps show how to set the COVID-19 database for the CS564 finall project.
1. Log into your MYSQL server
2. run CREATE DATABASE covid19
3. Create tables:
  CREATE TABLE county (name VARCHAR(255),
                       state VARCHAR(255),
                       total_deaths BIGINT,
                       population BIGINT,
                       PRIMARY KEY (name, state));
  CREATE TABLE cases (case_id VARCHAR(255),
                      occurred_at DATE,
                      daily_deaths BIGINT,
                      confirmed BIGINT,
                      name VARCHAR(255),
                      state VARCHAR(255),
                      PRIMARY KEY (case_id));
  CREATE TABLE socioeconomic (socio_id VARCHAR(255),
                      income VARCHAR(255),
                      life_expectancy BIGINT,
                      primary_physicians BIGINT,
                      mental_health_providers BIGINT,
                      uninsured BIGINT,
                      PRIMARY KEY (socio_id));
  CREATE TABLE hold (socio_id VARCHAR(255),
                     name VARCHAR(255),
                     state VARCHAR(255),
                     FOREIGN KEY (name, state) REFERENCES county(name, state),
                     PRIMARY KEY (socio_id, name, state));
                     //If you get an error regarding foreign key constraint during load data: first SET FOREIGN_KEY_CHECKS = 0;
                     //After you loaded data SET FOREIGN_KEY_CHECKS = 1;
