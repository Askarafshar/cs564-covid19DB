Following steps show how to set the COVID-19 database for the CS564 finall project.
1. Log into your MYSQL server
2. run CREATE DATABASE covid19
3. Create tables:
  CREATE TABLE county (name VARCHAR(255),
                       state VARCHAR(255),
                       total_deaths BIGINT,
                       population BIGINT,
                       PRIMARY KEY (name, state));
  CREATE TABLE cases (case_id BIGINT unsigned zerofill not null auto_increment,
                      occured_at VARCHAR(255),
                      daily_deaths BIGINT(255),
                      confirmed BIGINT,
                      name VARCHAR(255),
                      state VARCHAR(255),
                      PRIMARY KEY (case_id));
  CREATE TABLE socioeconomic (socio_id BIGINT unsigned zerofill not null auto_increment,
                      income VARCHAR(255),
                      life_expectancy BIGINT,
                      primary_physicians BIGINT,
                      mental_health_providers BIGINT,
                      uninsured BIGINT,
                      PRIMARY KEY (socio_id));
  CREATE TABLE hold (name VARCHAR(255),
                       state VARCHAR(255),
                       socio_id BIGINT,
                       PRIMARY KEY (name, state,socio_id));
4. Load .csv data into tables:
  SET GLOBAL local_infile = true;
  mysql -u <your username> -p --local-infile covid19
  LOAD DATA LOCAL INFILE '<PATH TO YOUR .CSV FILE>/County_table.csv' 
                                                 INTO TABLE county FIELDS TERMINATED BY ',' 
                                                 LINES TERMINATED BY '\n'; [LINUX COMMAND!]
5. Repeat step 4 for all tables with their own table.
